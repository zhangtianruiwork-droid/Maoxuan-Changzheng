## 设计理念

### 1. 调查前置

毛泽东最著名的方法论是"没有调查就没有发言权"。原版 skill 没有认真对待这一点——它在没有了解你的实际情况的前提下直接套框架。

v2.0 每次对话先进入**调查摸底阶段**，问3-5个关键问题，了解真实情况，然后才分析。

### 2. 矛盾重构

毛泽东最强大的能力之一，是**质疑别人的问题框架**。他不是接受你给出的问题，然后解答；他是首先判断你的问题对不对，你找的主要矛盾是不是真的主要矛盾。

v2.0 内置了**矛盾重构框架**，专门处理"用户说的问题往往不是真正的主要矛盾"这个现实。

### 3. 毛式苏格拉底法

毛泽东教人的方式，不是告诉你答案，而是通过追问让你自己想清楚。他的著作里大量使用"这难道不是...吗？"、"你有没有想过...？"这类句式。

v2.0 内置了**反问追问库**和**苏格拉底连环追问**结构，真正把这种对话方式落地。

### 4. 错误模式识别

毛泽东对各种错误思想有精准的命名和批判：教条主义、冒险主义、投降主义、官僚主义、尾巴主义。

v2.0 能识别用户落入这些模式的信号，并指出来。

## 效果对比示例

**用户**：我们是个小创业公司，竞争对手比我们大很多，怎么办？

**v1.0 风格**：
> 农村包围城市，同志。先找根据地，从边缘场景突破，不要在对方主场硬打...

**v2.0 风格**：
> 等一下，先别急着给你策略。我先要了解情况。
>
> 第一，你的竞争对手"大很多"——大在哪里？是用户量大？资金大？还是品牌认知大？这三种"大"，含义完全不同，对策也不同。
>
> 第二，你说你们是"小创业公司"，你们有没有一块已经站稳的市场——哪怕很小——在那里你们是比对手做得更好的？还是现在完全没有根据地，在整个市场里追着对手跑？
>
> 第三，你的主要矛盾是什么？是"打不过对手"，还是"还没找到自己的根据地"？这是两个不同的问题。
